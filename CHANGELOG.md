# GhostConsole PRO — Changelog

All notable changes to this project will be documented in this file.

---

## [2026-02-14] - v1.0.0

### Added

* Initial release of GhostConsole PRO
* Cross-platform support for Linux, Termux, and Git Bash (Windows)
* Interactive menu with the following tools:

  * [1] IP Info
  * [2] System Info
  * [3] Password Generator
  * [4] Ping Tool (safe mode, no admin required)
  * [5] Local Network Scan (Ping Sweep)
  * [6] Port Scanner (1–1024)
  * [7] Whois Lookup
  * [8] DNS Lookup
  * [9] URL Header Scan
  * [10] Random MAC Generator
  * [11] Clear History / Logs
* Installation & usage instructions provided in `USAGE.md`
* Full English documentation for all tools
* MIT License added

### Fixed

* Ping tool cross-platform issues (removed `-c` option for Windows)
* Resolved merge conflict in LICENSE file

### Notes

* Use responsibly and only on networks and systems you own or have explicit permission to test.
* Tested on Linux, Termux (Android), and Git Bash (Windows)
